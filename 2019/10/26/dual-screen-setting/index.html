ubuntu 双屏设置问题

Posted on 2019-10-26 | In -ubuntu |

问题：双屏设置后重启每次开机会失效,网上有看到通过这个设置成功

Auto-detect connected display hardware and load appropiate X11 setup using xrandr

这个东西可以保存一些屏幕的配置。

第一步，安装软件

sudo apt install autorandr
第二步，将屏幕调整到你想要的最佳状态。

设置-》设备-》显示

第三步，保存屏幕配置

这一步是要生成第二步弄好的屏幕设置文件。

autorandr -s best
best是配置文件的名字。

第四步，将生成的配置文件设置为默认的配置文件

autorandr -d best
